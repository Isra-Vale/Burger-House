# 🍔 Burger House — Frontend

A modern React frontend for a burger restaurant, focused on delivering a smooth ordering experience with a clean UI and responsive design.


## Features

* Modern UI with smooth animations
* Fully responsive (mobile → desktop)
* Shopping cart with instant updates
* Authentication flow (UI + validation)
* Admin dashboard with CRUD interface
* Search and filter burgers
* Contact form with validation
* Custom loading animation (ingredient fall)


## Tech Stack

* React 18
* Vite
* Tailwind CSS
* Framer Motion
* React Router v6
* React Hook Form + Zod
* TanStack Query
* Axios


## Getting Started

npm install
npm run dev

App runs at:
http://localhost:5173
